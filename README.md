#Inventory Management System
A Flask REST API with OpenFoodFacts integration.

#The setup
1. Install dependencies: `pip install flask requests`
2. Run the API: `python app.py`
3 .Run the Admin Portal: `python cli_interface.py`
4. Run Tests: `python testing.py`

#Features
- Full CRUD for inventory items.
- Barcode lookup via OpenFoodFacts API.
- CLI Administrator interface.
- Unit testing suite.
